# Sales Analytics System

A comprehensive Python system for processing, analyzing, and reporting sales transaction data with data quality validation, API integration, and business analytics.

## Overview

This assignment builds a Sales Data Analytics System that processes sales data, integrates with external APIs, performs analysis, and generates reports. The system tests understanding of Python fundamentals including data structures, functions, file operations, API integration, and error handling.


## Quick Start

### Run Interactive Application
```bash
python main.py
```

## Dependencies

```
requests>=2.31.0
```

Install with:
```bash
pip install -r requirements.txt
```

## Installation & Setup

1. Clone or download the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Ensure `sales_data.txt` is in the `data/` folder
4. Run the application: `python main.py`

# Clone the project
git clone https://github.com/yourusername/sales-analytics-system.git
cd sales-analytics-system

# Install dependencies
